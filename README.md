# AI Academic Assistant (Demo)

A lightweight demo of an AI-powered academic assistant designed to help students interact with study materials more effectively.

## 🚀 Features

* Upload and process academic documents (PDF)
* Ask questions based on document content
* Generate summaries using AI

## 🧠 Concept

This project explores how AI can improve student productivity by turning static materials into interactive learning tools.

## 🛠️ Tech Stack

* Node.js
* Express
* OpenAI API (or any model you're using)
* PDF parsing tools

## ⚠️ Note

This is a simplified public demo version. The full project contains more advanced features and optimizations.

## 📌 Status

Actively in development.

## ▶️ Run Locally

npm install  
npm start  

Server runs on http://localhost:3000
